# Athena AI

A modern Vue.js application built with the latest versions of Vue, PrimeVue, Tailwind CSS, and Vite.

## 🚀 Tech Stack

- **Vue.js 3.5** - The Progressive JavaScript Framework
- **PrimeVue 4.4** - Rich UI Component Library with Aura theme
- **Tailwind CSS 4.1** - Utility-First CSS Framework
- **Vite 7.1** - Next Generation Frontend Tooling
- **PrimeIcons 7.0** - Icon library

## 📦 Installation

```bash
# Install dependencies
npm install
```

## 🛠️ Development

```bash
# Start development server
npm run dev
```

The application will be available at `http://localhost:5173/`

## 🏗️ Build

```bash
# Build for production
npm run build
```

The build artifacts will be stored in the `dist/` directory.

## 👀 Preview

```bash
# Preview production build locally
npm run preview
```

## 📁 Project Structure

```
athena-ai/
├── public/              # Static assets
├── src/
│   ├── assets/         # Dynamic assets (images, etc.)
│   ├── components/     # Vue components
│   ├── App.vue        # Root component
│   ├── main.js        # Application entry point
│   └── style.css      # Global styles with Tailwind directives
├── index.html          # HTML entry point
├── vite.config.js      # Vite configuration
├── tailwind.config.js  # Tailwind CSS configuration
├── postcss.config.js   # PostCSS configuration
└── package.json        # Project dependencies
```

## 🎨 Features

- ✅ Vue 3 Composition API with `<script setup>` syntax
- ✅ PrimeVue UI components (Button, Card, InputText, DataTable, etc.)
- ✅ Tailwind CSS utility classes
- ✅ PrimeIcons integration
- ✅ Responsive design
- ✅ Modern build tooling with Vite
- ✅ Hot Module Replacement (HMR)
